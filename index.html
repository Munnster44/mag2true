True ⇄ Magnetic Calculator (Offline)

Uses WMM-2025 (World Magnetic Model) coefficients offline. Enter a coordinate, then apply declination to convert headings.
Model: WMM-2025 (built-in)
Sign convention used here: East declination = +, West declination = −. Conversions: Magnetic = True − Declination, True = Magnetic + Declination.

This app stores the imported model in your browser (localStorage). No internet required.
